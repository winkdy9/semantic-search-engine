# Semantic Search Engine

## О проекте

Реализация собственного индекса для поиска ближайших соседей в пространстве текстовых эмбеддингов.

Проект демонстрирует построение упрощённого аналога vector search engine: преобразование текста в эмбеддинги, индексацию векторов и эффективный поиск похожих объектов.

---

## Основные возможности

* генерация текстовых эмбеддингов;
* хранение векторных представлений;
* поиск k ближайших соседей;
* сравнение с полным перебором;
* оценка производительности.

## Реализация индекса

Индекс построен на основе KD-tree.

Каждый узел дерева хранит:

* embedding vector;
* идентификатор объекта;
* текстовый payload;
* координатную ось разбиения;
* ссылки на дочерние узлы.

Структура реализована самостоятельно без использования готовых библиотек поиска ближайших соседей.

---

## Алгоритм поиска

Для запроса:

1. вычисляется embedding;
2. выполняется обход KD-tree;
3. поддерживается max-heap из k ближайших элементов;
4. производится отсечение ветвей, которые не могут улучшить текущий результат.

---

## Сравнение с линейным поиском

Реализован baseline:

```
O(N)
```

который сравнивается с поиском через индекс.

Эксперимент проводится на размерах:

```
N = 100, 500, 1000, 5000, 10000
```

Метрики:

* время поиска;
* ускорение относительно линейного алгоритма.
